# React Admin Dashboard

This is a dummy project for me to learn how to create dashboard with React.

## Prerequisites

### Pre-Commit Hooks

Install at 2 stages:
- `commit-msg`: `pre-commit install --hook-type commit-msg`
- `pre-push`: `pre-commit install --hook-type pre-push`
